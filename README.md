# Smart Notes Pro

Smart Notes Pro is a modern React-based notes application built using Vite.
The project includes note management, dark/light mode, responsive UI, local storage support, and AI-inspired smart title generation.

---

# Live Demo

 https://smart-notes-app-eight.vercel.app
 
# Features

* Add Notes
* Delete Notes
* Search Notes
* Dark / Light Mode
* Responsive Modern UI
* Local Storage Persistence
* AI-Inspired Smart Title Generation


# AI Feature

The application includes a simple AI-inspired feature that automatically generates smart note titles based on the note content.

Example:

Input:
Learning React hooks and deployment using Vercel

Generated Title:
LEARNING REACT HOOKS

#  Installation

Clone the repository:

git clone https://github.com/niharikakhichi/smart-notes-app.git


Install dependencies:


npm install

Run development server:

npm run dev


Build for production:

npm run build
