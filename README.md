# Smart Notes Pro

A modern Notes Application built using React and Vite with component-based architecture, local storage support, dark/light mode, and responsive UI.


## Live Demo

 https://smart-notes-app-eight.vercel.app

# Features

* Add Notes
* Delete Notes
* Search Notes
* Dark / Light Mode
* Character Counter
* Responsive Design
* Local Storage
  
#Tech Stack

* React.js
* Vite
* JavaScript
* CSS3
* Local Storage API
