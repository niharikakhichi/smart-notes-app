# Smart Notes Pro

A modern Notes Application built using React and Vite with component-based architecture, local storage support, dark/light mode, and responsive UI.


## Live Demo

 https://smart-notes-app-eight.vercel.app



## Features

* Add Notes
* Delete Notes
* Search Notes
* Dark / Light Mode
* Character Counter
* Responsive Design
* Local Storage Persistence
* Modern Glassmorphism UI



## Tech Stack

* React.js
* Vite
* JavaScript (ES6+)
* CSS3
* Local Storage API

## Installation

Clone the repository:

git clone https://github.com/niharikakhichi/smart-notes-app.git


Install dependencies:

npm install

Run development server:

npm run dev


Build for production:

npm run build
